TITLE: Jeu Memory avancé
Description: Jeu de memoire où le but est de trouver les pairs d'émojis semblable avec la présence de cartes spéciales qui peuvent être soit bénéfiques soit nocives. 
Jouer à des difficultés différentes et faites de votre mieux pour obtenir le meilleur score parmi vos amis.

Fonctionnalités principales: Créer un environnement acceuillant interactive ou l'utilisateur choisit la difficulté du jeu et commence. 

Coder en html css et java contenant des fonctionalités multiples pour assurer le bon fonctionnement du projet.

Lien vers la page GitHub Pages (rendu final): https://amine-kharrat.github.io/Amine-kharrat-Memory-card-Avanc-/

Nouveautés explorées : Comment bien manipuler les objets grâce au java script et le dom et assurer l'harmonie entre différentes fonction du code.
Difficultés rencontrées : Créativité du code et la facon la plus adécquate pour créer le gameboard et optimiser la sélection des cartes. Ainsi que le state du jeu qui prend en charge le déroulement du jeu.
Solutions apportées : Divers cartes spéciales qui rend le jeu plus amusant, utilisation du DOM pour bien manipuler l'espace du jeu et le nombre de cartes selon la difficulté choisit. 
UNe variable state qui contient l'état actuelle du jeu accompagné de divers fonctions qui asuurent l'update du site comme le temps score et best score.
